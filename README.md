# FluffyCustomDrops

ปลั๊กอิน Minecraft สำหรับเพิ่มรางวัลพิเศษจากการทุบบล็อก เก็บพืช และตีมอนสเตอร์

## คำสั่ง

```
/fluffydrops reload
/fdrops reload
/fd reload
```

## Permissions

```
fluffydrops.reload
```

สิทธิ์ในการใช้คำสั่ง reload (default: op)

## การตั้งค่า

แก้ไขไฟล์ `config.yml` เพื่อปรับแต่งรางวัล

- **chance**: โอกาสในการได้รับรางวัล (1.0 = 100%, 0.1 = 10%, 0.01 = 1%)
- **command**: คำสั่งที่จะรัน (ใช้ %player% สำหรับชื่อผู้เล่น)

## ข้อความ

แก้ไขไฟล์ `message.yml` เพื่อปรับแต่งข้อความ

รองรับ Hex Color: `&#FF69B4` และ Color Code: `&a`, `&c`
